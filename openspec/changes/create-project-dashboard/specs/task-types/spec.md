## ADDED Requirements

### Requirement: 任务类型定义
系统SHALL支持自定义任务类型及其颜色配置。

#### Scenario: 任务类型创建
- **WHEN** 管理员创建新任务类型时
- **THEN** 支持定义类型名称、颜色、描述
- **AND** 颜色选择器提供常用颜色选项
- **AND** 支持设置类型是否启用

#### Scenario: 任务类型编辑
- **WHEN** 管理员修改现有任务类型时
- **THEN** 支持修改类型名称、颜色、描述
- **AND** 修改后自动更新所有相关任务显示
- **AND** 保留历史类型信息用于追溯

#### Scenario: 任务类型删除
- **WHEN** 管理员删除任务类型时
- **THEN** 提示确认删除操作
- **AND** 提供任务迁移选项（迁移到其他类型）
- **AND** 删除后不影响历史数据展示

### Requirement: 任务类型应用
系统SHALL在甘特图中根据任务类型显示相应颜色。

#### Scenario: 甘特图颜色映射
- **WHEN** 任务在甘特图中显示时
- **THEN** 根据任务类型自动应用对应颜色
- **AND** 颜色显示应清晰可辨
- **AND** 支持色盲友好的颜色方案

#### Scenario: 类型筛选功能
- **WHEN** 用户需要筛选特定类型任务时
- **THEN** 支持按任务类型进行筛选
- **AND** 支持多选类型同时显示
- **AND** 筛选状态应持久化保存
