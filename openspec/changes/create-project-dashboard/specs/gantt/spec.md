## ADDED Requirements

### Requirement: 可滑动日期甘特图
系统SHALL提供可滑动的日期甘特图，支持不同任务类型的可视化展示。

#### Scenario: 滑动浏览时间轴
- **WHEN** 用户查看甘特图时
- **THEN** 支持左右滑动浏览不同日期范围
- **AND** 支持缩放功能（周视图/月视图）
- **AND** 加载时默认显示北京日期当天

#### Scenario: 当前日期红线标示
- **WHEN** 甘特图加载完成
- **THEN** 在当天日期位置显示垂直红线
- **AND** 红线应清晰可见，贯穿整个甘特图高度
- **AND** 红线位置随日期自动更新

#### Scenario: 任务类型颜色显示
- **WHEN** 甘特图显示任务时
- **THEN** 根据任务类型显示不同颜色
- **AND** 颜色配置应支持自定义
- **AND** 任务条上显示开发人员、测试人员信息

#### Scenario: 任务信息显示
- **WHEN** 任务条在甘特图中显示时
- **THEN** 默认显示任务名称、开发人员、测试人员
- **AND** 显示阶段起始日期和当日进展（如果有）
- **AND** 空字段自动隐藏，保持界面简洁

### Requirement: 任务编辑功能
系统SHALL支持通过双击任务进行快速编辑。

#### Scenario: 双击任务编辑
- **WHEN** 用户双击甘特图中的任务条
- **THEN** 弹出任务编辑模态框
- **AND** 显示当前任务的所有可编辑字段
- **AND** 支持修改进度、状态、备注等信息

#### Scenario: 实时保存修改
- **WHEN** 用户在编辑模态框中修改任务信息
- **THEN** 点击保存后立即更新甘特图显示
- **AND** 自动记录修改历史（修改人、修改时间、修改内容）
- **AND** 无需审批流程，实现极速模式
