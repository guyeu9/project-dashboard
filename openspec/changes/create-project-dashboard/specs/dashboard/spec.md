## ADDED Requirements

### Requirement: 项目全景仪表板
系统SHALL提供一个多项目聚合的仪表板视图，显示全公司项目的宏观状态信息。

#### Scenario: 显示核心指标
- **WHEN** 用户访问仪表板首页
- **THEN** 显示并行项目数量、延期/风险项目数量、待开始项目数量
- **AND** 指标数据应实时自动计算更新

#### Scenario: 项目状态筛选
- **WHEN** 用户选择状态筛选器
- **THEN** 仪表板仅显示符合筛选条件的项目
- **AND** 支持按开发联调、测试联调、上线状态筛选

### Requirement: 聚合甘特图展示
系统SHALL以项目为单位在甘特图中展示项目时间轴和进度状态。

#### Scenario: 显示项目时间轴
- **WHEN** 用户查看聚合甘特图
- **THEN** Y轴显示项目名称列表
- **AND** X轴显示时间轴（支持周/月视图切换）
- **AND** 每个项目显示起止时间的进度条

#### Scenario: 进度状态颜色标识
- **WHEN** 项目进度正常时
- **THEN** 进度条显示为绿色
- **WHEN** 项目延期或包含阻塞任务时
- **THEN** 进度条显示为红色

#### Scenario: 悬浮详情展示
- **WHEN** 用户鼠标悬停在项目进度条上
- **THEN** 显示项目备注信息（合作方、开发人员、测试人员、负责人）
- **AND** 信息应简洁明了，便于快速了解项目背景

#### Scenario: 项目详情下钻
- **WHEN** 用户点击某个项目进度条
- **THEN** 跳转到该项目的详情甘特图页面
- **AND** 保留当前筛选和时间范围状态
