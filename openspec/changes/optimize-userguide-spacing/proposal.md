# 优化使用说明页面间距提案

## Context
用户反馈使用说明页面（UserGuide）空白太多，需要更紧凑，减少卡片之间和内容之间的空白。当前页面布局过于松散，影响用户阅读体验和信息密度。

## Scope
- 涉及文件：`src/pages/UserGuide/index.tsx` 和 `src/pages/UserGuide/index.css`
- 影响范围：使用说明页面的所有卡片、内容区块和元素间距

## Behavior
1. **卡片间距优化**：
   - 头部卡片与内容卡片之间的间距从24px减少到16px
   - 内容卡片之间的间距从24px减少到16px

2. **卡片内部间距优化**：
   - 调整Space组件的size属性，从large改为middle
   - 减少功能区块（overview、detail、steps、tips、notes）的内边距从16px到12px
   - 减少区块header的底部外边距从16px到12px
   - 减少段落的底部外边距从16px到12px

3. **分隔线优化**：
   - 调整Divider组件的样式，减少其上下间距

## Risks
- 可能影响页面的可读性，如果间距调整过小
- 可能破坏现有的视觉层次结构
- 需要确保在不同屏幕尺寸下都能保持良好的显示效果

## 验收标准
1. 页面整体布局更紧凑，减少不必要的空白
2. 卡片之间的间距统一且适中
3. 卡片内部内容间距合理，不影响可读性
4. 响应式设计保持良好，在移动端也能正常显示
5. 视觉层次结构清晰，重要信息突出
