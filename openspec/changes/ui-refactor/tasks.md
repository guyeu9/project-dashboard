## 任务1：创建统一的CSS样式变量文件

**描述**：创建统一的CSS样式变量文件，定义色彩、字体、间距等设计系统变量

**具体要求**：
1. 创建src/styles/variables.css文件
2. 定义色彩系统变量，包括主色调、辅助色和中性色
3. 定义字体系统变量，包括字体大小、字重和行高
4. 定义间距系统变量，包括内边距、外边距和组件间距
5. 定义边框和阴影变量

**验证方法**：
- CSS变量文件创建成功
- 所有设计系统变量定义完整
- 变量命名规范一致
